## A

### argument

Un argument est une valeur fournie à une [fonction](#fonction) ou à une [méthode](#méthode) lors de son appel. Les arguments sont utilisés pour passer des données à une fonction ou à une méthode afin qu'elle puisse les utiliser pour effectuer une tâche spécifique.

## C

### compilateur

Un compilateur est un programme informatique qui traduit du code source écrit dans un langage de programmation en code machine exécutable par un ordinateur.
Contrairement à un [interpréteur](#interpréteur), un compilateur traduit tout le code source en une seule fois avant de l'exécuter.

## F

### fonction

Une fonction est un bloc de code qui effectue une tâche spécifique lorsqu'elle est appelée. Les fonctions sont utilisées pour organiser le code en le divisant en tâches plus petites et plus gérables. Les fonctions peuvent accepter des [arguments](#argument) et renvoyer des valeurs.
Plusieurs fonctions sont déjà disponibles dans Python, comme `print()` ou `len()`, mais il est possible de créer ses propres fonctions.

## I

### interpréteur

Un interpréteur est un programme informatique qui lit et exécute du code source écrit dans un langage de programmation. L'interpréteur Python est un programme qui lit et exécute du code source écrit en Python.
Vous pouvez ouvrir l'interpréteur Python en tapant `python` ou `python3` dans votre terminal.
Vous optiendrez un [prompt](#prompt) `>>>` qui vous permettra d'écrire du code Python.

### instruction

Une instruction est une commande ou une action spécifique qui est exécutée par un programme informatique. Les instructions sont utilisées pour définir le comportement d'un programme et pour effectuer des tâches spécifiques, comme l'affectation de valeurs à des variables ou l'appel de fonctions.

### IDE

Un IDE (Integrated Development Environment) est un environnement de développement intégré qui regroupe plusieurs outils pour faciliter l'écriture, le débogage et le test de programmes informatiques. Les IDE sont souvent utilisés par les développeurs pour améliorer leur productivité et leur efficacité lors de l'écriture de code.
Exemple d'IDE populaires pour Python : PyCharm, Visual Studio Code

## M

### méthode

Une méthode est une fonction qui est associée à un objet spécifique. Les méthodes sont utilisées pour effectuer des opérations sur un objet ou pour interagir avec lui. En Python, les méthodes sont appelées en utilisant la notation pointée, par exemple `objet.methode()`.

## P

### prompt

Le prompt est un symbole ou un texte affiché par un [interpréteur](#interpréteur) pour indiquer qu'il est prêt à recevoir des commandes. Dans l'interpréteur Python, le prompt est généralement `>>>`.

## S

### script

Un script est un fichier contenant une suite d'[instructions](#instruction) qui sont exécutées les unes après les autres. Les scripts sont utilisés pour automatiser des tâches répétitives ou pour exécuter des programmes plus complexes. En Python, les scripts sont généralement enregistrés avec l'extension `.py`.

## T

### terminal

Un terminal est un programme informatique qui permet aux utilisateurs d'interagir avec un ordinateur en utilisant une interface en ligne de commande. Les utilisateurs peuvent saisir des commandes dans un terminal pour exécuter des programmes, naviguer dans le système de fichiers, et effectuer d'autres tâches.
